## Revenue Model Overview

### Business Model: B2B SaaS Subscription

| Element | Details |
|---------|---------|
| **Model Type** | Per-employee-per-month (PEPM) subscription |
| **Billing** | Monthly or annual (prepaid) |
| **Pricing** | $6-8 PEPM (annual/monthly) |
| **Target ARPA** | $100-150/month |
| **Contract** | Month-to-month (no lock-in) |

### Revenue Formula

```
Monthly Revenue = Active Customers x Average Employees x PEPM Price

Example:
30 customers x 20 avg employees x $7 PEPM = $4,200 MRR
```

## Break-Even Analysis

### Monthly Fixed Costs (Estimated)

| Cost | Amount |
|------|--------|
| Hosting (Railway) | $20 |
| Email (Resend) | $0-20 |
| Tools/subscriptions | $100 |
| LinkedIn Navigator | $100 |
| Domain/misc | $30 |
| **Total** | **$250-300** |

### Break-Even Calculation

```
Break-even MRR = Fixed Costs / Gross Margin
Break-even MRR = $300 / 0.85 = $353

At $115 ARPA:
Break-even customers = $353 / $115 = 4 customers
```

**Break-even: 4 customers at $115 ARPA**
