> **Purpose:** Track financial health of TeamPulse. Focus on metrics that matter for bootstrapped SaaS.
>

## North Star Financial Metric

**Monthly Recurring Revenue (MRR):** $0 --> $3-5K (6-month target)

**Why MRR:** It's the single number that captures growth, pricing power, and customer success for a subscription business.

## MRR Movement Tracking

### Monthly MRR Bridge

| Component | Formula | M3 | M4 | M5 | M6 |
|-----------|---------|----|----|----|----|
| Starting MRR | Previous month's ending | | | | |
| + New MRR | From new customers | | | | |
| + Expansion MRR | Upgrades, more seats | | | | |
| - Contraction MRR | Downgrades | | | | |
| - Churned MRR | Lost customers | | | | |
| = Ending MRR | Sum of above | | | | |

### MRR Movement Ratios

| Ratio | Formula | Target | Current |
|-------|---------|--------|---------|
| Quick Ratio | (New + Expansion) / (Churn + Contraction) | >4 | |
| Gross MRR Churn | Churned MRR / Starting MRR | <5% | |
| Net MRR Churn | (Churned - Expansion) / Starting MRR | <0% | |

## Benchmarks Reference

### SaaS Benchmarks (SMB Focus)

| Metric | Poor | Good | Great |
|--------|------|------|-------|
| MoM MRR Growth | <10% | 15-25% | >30% |
| Gross Margin | <70% | 75-85% | >85% |
| Net Revenue Retention | <90% | 100-110% | >120% |
| LTV:CAC | <3:1 | 3-5:1 | >5:1 |
| CAC Payback | >18 mo | 12-18 mo | <12 mo |
| Logo Churn | >10% | 5-10% | <5% |

### Rule of Thumb

- **Rule of 40:** Growth rate + Profit margin should be >40%
- **T2D3:** Triple revenue 2 years, then double 3 years (for VC-scale)
- **Bootstrapped target:** 20%+ annual growth, profitable

---

## DO NOT TRACK (Vanity Metrics)

- Total revenue (all-time) - Use MRR instead
- "Users" without paying distinction
- Gross merchandise value (unless marketplace)
- Revenue projections without assumptions
- Anything you can't act on
