# Marketing Manager Agent: Landing Page Strategy
## Landing Page Strategy & Copy

> **Purpose**: Convert visitors into leads/trials with a high-converting landing page.
> **Update Frequency**: Iterate based on conversion data (minimum monthly)
> **Feeds Into**: Lead Gen, Sales Discovery

## 2. Hero Section Copy Framework

### Headline Formulas

**Formula 1: End Result**
```
[Achieve X Result] Without [Pain Point]
Example: "Book 20+ Sales Calls Weekly Without Cold Calling"
```

**Formula 2: For + Outcome**
```
[Outcome] for [Specific Audience]
Example: "AI-Powered Lead Gen for Bootstrapped SaaS Founders"
```

**Formula 3: Time-Bound Result**
```
[Result] in [Timeframe]
Example: "Double Your Pipeline in 30 Days"
```

**Formula 4: Eliminate Pain**
```
Stop [Painful Activity]. Start [Desired Activity].
Example: "Stop Manual Prospecting. Start Closing Deals."
```

### Your Hero Section

**Headline:**
```
[Your headline here]
```

**Subheadline:** (Expand on headline, add specificity)
```
[Your subheadline here - explain HOW or add credibility]
```

**Primary CTA Button:**
```
[CTA text - action-oriented, specific]
Examples: "Start Free Trial" / "Book a Demo" / "Get Started Free"
```

**Supporting Element:**
```
[ ] Product screenshot/mockup
[ ] Demo video (autoplay, muted)
[ ] Animated GIF of product
[ ] Illustration
```

**Trust Element Below CTA:**
```
Examples:
- "No credit card required"
- "14-day free trial"
- "Used by 500+ teams"
- "Setup in 5 minutes"
```

## 4. Problem Section Copy

### Structure
```
**Header:** "If you're like most [ICP role]..."

**Pain Point 1:** [Specific painful situation]
- How it manifests
- What it costs them

**Pain Point 2:** [Second pain point]
- How it manifests  
- What it costs them

**Pain Point 3:** [Third pain point]
- How it manifests
- What it costs them

**Transition:** "It doesn't have to be this way."
```

### Example:
```
If you're like most B2B founders...

You're spending 10+ hours every week on manual prospecting
- Researching companies on LinkedIn
- Finding contact info
- Writing personalized messages
- Getting ignored anyway

Your pipeline is unpredictable
- Some weeks are feast, others famine
- You can't forecast revenue reliably
- Growth feels random, not systematic

You're too busy to build a scalable system
- You know you need to fix this
- But there's always something more urgent
- Meanwhile, competitors are scaling faster

It doesn't have to be this way.
```

## 7. Full Testimonial Section

### Testimonial Collection Template

Ask customers these questions:
1. What was your situation before using [product]?
2. What specific results have you achieved?
3. What would you tell someone considering [product]?

### Testimonial Display Format
```
"[Quote - focus on specific results, not just praise]"

[Photo]
[Full Name]
[Role], [Company]
[Optional: specific metric or result]
```

### Testimonial Priorities
1. **Results-focused** > praise-focused
2. **Specific** > generic
3. **With photo** > without
4. **With company** > anonymous
5. **Video** > text (if you have them)

## 9. CTA Strategy

### Primary CTA
```
Text: [Action-oriented verb + outcome]
Examples:
- "Start Free Trial"
- "Book a Demo"
- "Get Started Now"
- "Claim Your Free Account"

Button Color: [High contrast to page]
Placement: Hero, after testimonials, footer
```

### Secondary CTA (Lower commitment)
```
Text: [Lower-friction action]
Examples:
- "Watch Demo"
- "See How It Works"
- "View Pricing"
- "Download Guide"
```

### CTA Best Practices
- [ ] One clear primary CTA per screen
- [ ] Button text describes what happens when clicked
- [ ] Surrounded by supporting trust elements
- [ ] Above the fold AND at page bottom
- [ ] Consistent throughout page

## 10. Landing Page Checklist

### Before Launch
- [ ] Headline passes the 5-second test (clear value)
- [ ] CTA visible above the fold
- [ ] Mobile responsive
- [ ] Page loads in <3 seconds
- [ ] All links work
- [ ] Analytics/tracking installed
- [ ] Social proof is real/verifiable
- [ ] No spelling/grammar errors
- [ ] Form works and leads to right place

### Conversion Optimization
- [ ] Single, focused CTA
- [ ] Remove navigation (for dedicated landing pages)
- [ ] Trust badges visible
- [ ] Testimonials with photos/names
- [ ] Risk reversal (guarantee, free trial)
- [ ] Urgency/scarcity if genuine

## 11. A/B Testing Priorities

Test these elements in order of impact:

1. **Headline** (Biggest impact)
2. **CTA button text and color**
3. **Social proof placement**
4. **Hero image vs video**
5. **Page length** (short vs long)
6. **Pricing display**

### Testing Framework
```
Test: [What you're testing]
Hypothesis: [Why you think B will beat A]
Metric: [What you're measuring]
Duration: [Minimum 2 weeks or 100 conversions per variant]
Result: [Winner + lift percentage]
Learning: [What this teaches us]
```
